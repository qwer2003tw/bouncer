### `extract_exit_code()` 函數

```python
def extract_exit_code(result: str) -> int:
    """Extract exit code from command result string.

    Result format:
        "command output\n\n---\nExit code: 0"

    Returns:
        int: exit code (0 = success, non-zero = failure)
    """
    match = re.search(r'Exit code: (\d+)', result)
    if match:
        return int(match.group(1))
    return -1  # fallback
```

**已驗證**：此函數已在 trust session path 使用，測試覆蓋充分。
