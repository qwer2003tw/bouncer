## Technical Context

### Callback Flow

```
Telegram callback button click
  → API Gateway → Lambda (callbacks.py)
  → handle_XXX_callback()
  → [immediate feedback?] update_message()
  → [actual operation] start_deploy() / execute_command()
  → [completion notification] send_telegram_message()
```

```python
def test_execute_callback_immediate_feedback(mocker):
    """Execute callback updates message immediately."""
    mock_update = mocker.patch('src.callbacks.update_message')
    mock_execute = mocker.patch('src.callbacks.execute_command')

    handle_execute_callback(
        callback_data='...',
        callback_query_id='test_query',
        message_id=12345,
        chat_id=67890
    )

    # Assert update_message was called first
    mock_update.assert_called_once_with(
        chat_id=67890,
        message_id=12345,
        text="⚙️ 正在執行...",
        parse_mode='Markdown'
    )

    # Assert execute_command was called after
    assert mock_execute.called
    # Verify call order
    assert mock_update.call_count == 1
```
