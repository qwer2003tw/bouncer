## Technical Context

### Telegram Message Formatting

Telegram 支援兩種格式化方式：

#### 1. Parse Mode (MarkdownV1/V2)
```python
send_telegram_message(
    chat_id=chat_id,
    text="**bold** _italic_",
    parse_mode='MarkdownV2'
)
```

**Drawback**: 需要 escape 許多特殊字元（`_*[]()~`#+-=|{}.!`）

#### 2. Entities Mode
```python
send_telegram_message(
    chat_id=chat_id,
    text="bold italic",
    entities=[
        {'type': 'bold', 'offset': 0, 'length': 4},
        {'type': 'italic', 'offset': 5, 'length': 6},
    ]
)
```

**Advantage**: 不需要 escape，直接指定格式範圍。

### Expandable Blockquote Entity

Telegram Bot API 5.0+ 支援 `expandable_blockquote` entity type：

```python
{
    'type': 'expandable_blockquote',
    'offset': 0,
    'length': len(text)
}
```

**行為**：
- 初始狀態：collapsed（只顯示前幾行）
- 點擊後：expanded（顯示完整內容）
- 再點擊：collapsed

### Phase 1: Entities builder

**File**: `src/telegram_entities.py`

**New function**:

```python
def format_command_output(output: str, threshold: int = 50) -> tuple[str, list]:
    """Format command output with expandable blockquote if long.

    Args:
        output: Raw command output
        threshold: Line count threshold for using expandable blockquote

    Returns:
        (text, entities): Formatted text and Telegram entities
    """
    # Handle empty output
    if not output or output.strip() == '':
        return '(no output)', []

    # Count lines
    lines = output.split('\n')
    line_count = len(lines)

    if line_count <= threshold:
        # Short output: no special formatting
        return output, []

    # Long output: expandable blockquote
    entities = [
        {
            'type': 'expandable_blockquote',
            'offset': 0,
            'length': len(output)
        }
    ]
    return output, entities
```

from src.telegram_entities import format_command_output

def send_execute_result(chat_id, command, output, exit_code):
    icon = '✅' if exit_code == 0 else '❌'

    # Format output with expandable blockquote if long
    formatted_output, entities = format_command_output(output)

    message = f"{icon} Command: {command}\n\n{formatted_output}\n\nExit code: {exit_code}"

    # Adjust entity offsets (skip header)
    header_len = len(f"{icon} Command: {command}\n\n")
    for entity in entities:
        entity['offset'] += header_len

    send_telegram_message(chat_id=chat_id, text=message, entities=entities)
```

def test_format_command_output_short():
    """Short output (<50 lines) has no blockquote."""
    output = '\n'.join([f'line {i}' for i in range(10)])
    text, entities = format_command_output(output)

    assert text == output
    assert entities == []

def test_format_command_output_long():
    """Long output (>50 lines) has expandable blockquote."""
    output = '\n'.join([f'line {i}' for i in range(60)])
    text, entities = format_command_output(output)

    assert text == output
    assert len(entities) == 1
    assert entities[0]['type'] == 'expandable_blockquote'
    assert entities[0]['offset'] == 0
    assert entities[0]['length'] == len(output)

def test_format_command_output_empty():
    """Empty output returns placeholder."""
    text, entities = format_command_output('')
    assert text == '(no output)'
    assert entities == []

def test_format_command_output_threshold():
    """Custom threshold works."""
    output = '\n'.join([f'line {i}' for i in range(30)])

    # threshold=50: no blockquote
    text, entities = format_command_output(output, threshold=50)
    assert entities == []

    # threshold=20: blockquote
    text, entities = format_command_output(output, threshold=20)
    assert len(entities) == 1
    assert entities[0]['type'] == 'expandable_blockquote'
```
