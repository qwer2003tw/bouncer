# S36-001: Plan

## Architecture Decision
**用 GitHub API 取 template.yaml 新舊版本做比對**，不依賴 git binary（Lambda 環境沒有）。

- 舊版本：從 DDB `template_s3_url` 下載（已有最新 deploy 的 packaged template，可 reverse-extract）
  → 問題：packaged template 已做 SAM transform，不是原始 template
- **更好：** GitHub API 取 previous commit 的 template.yaml vs 現在 HEAD 的 template.yaml

```python
# GitHub compare API
GET /repos/{owner}/{repo}/compare/{base}...{head}
# 取 files[].patch 找 template.yaml 的 diff
```
